# Texas Used Vehicle ROI Analysis
### South Central Texas Automotive Group | San Antonio, TX | 2026

## Business Problem
South Central Texas Automotive Group acquires, repairs, and resells used vehicles 
in the San Antonio market. This project analyzes the Texas used vehicle market to 
identify which makes, models, and price ranges deliver the highest return on 
investment (ROI) and fastest turnover, enabling data-driven acquisition decisions.

## Tools and Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn)
- **SQL** (MySQL)
- **Tableau** (Interactive Dashboard)
- **Jupyter Notebook**
- **GitHub** (Version Control)
